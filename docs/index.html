PowerBackup

Multi-Database Backup & Restore Tool with Beautiful Logging

A complete, enterprise-grade database backup and restore solution built with Node.js. Features beautiful emoji-enhanced logging, automated scheduling, GPG encryption, and support for MySQL and PostgreSQL.

📦 Get Started 📚 Documentation

✨ Key Features

🔄 Multi-Database Support

Full backup and restore support for MySQL and PostgreSQL databases with optimized dumping and restoration processes.

🎨 Beautiful Logging

Rich, emoji-enhanced logs with formatted timestamps and color-coded output for better visibility and debugging.

🔒 Security First

GPG encryption for backup files, credential protection, and secure file permissions to keep your data safe.

⏰ Automated Scheduling

Multiple deployment options including PM2, Systemd, Cron, and Windows Task Scheduler for automated hourly backups.

🧪 Test Restore

Safe restore to temporary database for verification, ensuring your backups are reliable and restorable.

📋 Table-Level Restore

Restore individual tables from backups with interactive CLI selection and comprehensive table listing.

🗜️ Compression

Gzip compression to significantly reduce backup file sizes while maintaining data integrity.

🛡️ Retention Policies

Automated cleanup of old backups based on configurable retention rules (hourly, daily, weekly, monthly, yearly).

📱 Cross-Platform

Full support for Windows, Linux, and macOS with native deployment options for each platform.

🚀 Quick Start

# Clone the repository
 git clone https://github.com/anonnion/powerbackup.git
 cd powerbackup

# Run setup script
./setup.sh

# Create your first backup
npm run backup test-db

# Test restore to temporary database
npm run test-restore test-db

# List available databases
npm run list

# Get help
npm run help
